NPV Telehealth Appts

Licensing Restrictions

Before scheduling, please check: If the patient resides outside of Pennsylvania, you need to make sure that you are appropriately licensed in that state unless that state has authorized out-of-state providers to conduct new patient telehealth across state lines without a license. Some states have only authorized telehealth for new patients for COVID related purposes without a license. Some states have created an expedited licensure process. For example, you must be licensed in New Jersey to provide non-COVID-related telemedicine services to new patients. New Jersey has created an expedited licensure process.

Refer to the updated State Licensure chart

Clinical Eligibility - Tips

  1. You should use your best judgment in determining whether the clinical circumstances warrant proceeding with a new patient telehealth visit.

    a. Determine that you can effectively care for the patient using the remote tools (Blue Jeans, Vidyo, telephone) without the quality of the evaluation being affected or compromised. e.g., can you appropriately evaluate a patient’s presenting complaint via a remote visit?

  2. You must still obtain and document the patient’s informed consent for telehealth services. Refer to the updated verbal consent tip sheet available on the Connected Health tab from the Penn Med Access Page at https://pennmedaccess.uphs.upenn.edu/ or http://connectedhealth.uphs.upenn.edu when already on the network.

  3. If visit is already underway when it is determined that care cannot be delivered effectively using telehealth services, you should suspend the phone and/or video visit and direct the patient to schedule an in-person appointment or to the emergency room, as appropriate. Note that the patient and/or payer should not be billed for this suspended visit.

Documentation

For visit documentation, the following EPIC SmartPhrase has been added to existing telemedicine documentation:

This visit does not require an in-person evaluation and can be conducted remotely via telehealth systems without adversely affecting the quality of care delivered.

Please use approved health system telemedicine documentation phrases including .COVTELEDOC, .COVVIDEO, and .COVTELEPHONE which will contain the above language.
